# Icons from web

![GitHub package.json version](https://img.shields.io/github/package-json/v/adereksisusanto/icons)

## Instalation

- **Via NPM**

  ```bash
  $ npm install @adereksisusanto/icons
  ```

- **Via Composer**
  ```bash
  $ composer require adereksisusanto/icons
  ```

## Use

```html
<!-- Online -->
<!-- All Icons -->

<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@adereksisusanto/icons@1.0.5/dist/css/all.icons.min.css"
/>

<!-- Local -->
<!-- All Icons -->
<link
  rel="stylesheet"
  href="node_modules/@adereksisusanto/icons/dist/css/all.icons.min.css"
/>
```
